# Dashboard Streamlit

Dashboard narrativo basado en los seis gráficos consolidados de `src/final_infographic.py`.

## Ejecutar

Desde la raíz del proyecto:

```powershell
cd app
streamlit run app.py
```

También funciona desde la raíz con:

```powershell
streamlit run app/app.py
```

La primera opción carga además el tema local definido en `app/.streamlit/config.toml`.
